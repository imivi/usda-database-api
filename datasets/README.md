# USDA Datasets CSV

After download the datasets from the USDA website, unzip them and place them in this directory:

```txt
datasets/
├── foundation/
|   ├── food.csv
|   ├── food_nutrient.csv
|   └── nutrient.csv
├── sr_legacy/
|   ├── food.csv
|   ├── food_nutrient.csv
|   └── nutrient.csv
└── fndds_survey/
    ├── food.csv
    ├── food_nutrient.csv
    └── nutrient.csv
```
